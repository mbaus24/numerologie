Bootstrap est une collection d'outils utiles à la création du design (graphisme, animation et interactions avec la page dans le navigateur, etc.) de sites et d'applications web. C'est un ensemble qui contient des codes HTML et CSS, des formulaires, boutons, outils de navigation et autres éléments interactifs, ainsi que des extensions JavaScript en option.

Installation 2 méthodes

Pour installer Boostrap, 2 méthodes s'offrent à nous:

Téléchargement direct

Voici le lien pour télécharger la denière version de Bootstrap (5.3)

Il suffit ensuite de placer les fichiers css et js dans les dossiers correspondants et de les relier à notre fichier html

Via CDN Conseillé

Les CDN sont des serveurs localisés dans tout le monde permettant un téléchargement rapide de petits fichiers

Pour installer Bootstrap de cette manière, il suffit de placer 2 lignes permettant le téléchargement du css et du js dans le < head>

Le tutoriel est détaillé ici et cela permet de mettre a jour Bootstrap plus facilement
